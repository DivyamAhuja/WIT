Functions


            Functions are block of code designed to perform a particular task.
            A Javascript function executes when 'something' invokes it (calls it).

            Syntax:

            function function_name(function_arguments){
                ...function_body
                return return_value //function may or may not return something
            }

            or ( ES6 syntax )

            const function_name = (function_arguments) => {
                ...function_body
                return return_value
            }

            Calling a function:

            function_name(args);
            or
            var result = function_name(args);

            example:
            const add = (a, b) => {
                return a+b;
            };
            This function will add a and b, and will return their sum.
